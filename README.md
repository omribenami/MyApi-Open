# MyApi

MyApi is a personal API platform with a web dashboard for managing:
- Service connectors (OAuth/API integrations)
- Token vault and scoped guest tokens
- Personas and identity docs
- Knowledge base documents
- Skills and marketplace items

## Project Structure
- `backend/` - API/backend services
- `frontend/` - frontend app code
- `src/` - current runtime implementation and dashboard app
- `docs/` - design and implementation documentation

## Local Development
```bash
npm install
npm run dev
```

## Notes
This repository should contain only MyApi project files.

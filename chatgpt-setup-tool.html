🤖 MyApi ChatGPT Setup Tool

Connect your MyApi account to ChatGPT in 5 minutes

1 Get Your MyApi Token

First, you need a MyApi token. This grants ChatGPT access to your connected services.

How to get your token:
  1. Open https://www.myapiai.com/dashboard/tokens
  2. Look for a token starting with myapi_
  3. Copy the entire token
  4. Paste it below
⚠️ Security: This tool runs entirely in your browser. Your token is never sent to any server except MyApi and ChatGPT. Clear your browser history if you want to remove stored tokens.
2 Create a Custom GPT

Open ChatGPT and follow these steps to create a new Custom GPT.

Step by step:
  1. Go to https://chatgpt.com/gpts/mine (requires ChatGPT Plus)
  2. Click "Create a GPT" button
  3. Name: MyApi Services
  4. Description: Access my connected OAuth services (Twitter, Google, Slack, Facebook, etc.)
  5. Click "Create"
  6. You should now be in the GPT editor
ℹ️ You need ChatGPT Plus ($20/month) to create Custom GPTs.
3 Add Instructions

In the ChatGPT editor, find the "Instructions" section and paste this:

In ChatGPT: Click the text area under "Instructions" (in the Configure tab), paste the text above, and scroll down to see "Actions".
4 Add API Action

In ChatGPT, scroll down to "Actions" section and add a new action.

Steps:
  1. Under "Actions", click "Create new action"
  2. Click "Import from URL"
  3. Paste this URL:
    https://raw.githubusercontent.com/omribenami/MyApi/main/openapi-chatgpt-schema.yaml
  4. Click "Import"
If the URL import doesn't work, you can manually paste the OpenAPI schema. Contact support for the raw schema.
5 Set Authentication

After importing the action, you'll see an "Authentication" section.

Configure authentication:
  1. Find the "Authentication" section in the action
  2. Select "API Key" from dropdown
  3. Set Auth Type to: Bearer
  4. Leave "API Key Name" blank
  5. Set "API Key Value" to your MyApi token (from Step 1)
⚠️ Keep this token private! Anyone with this token can access your connected OAuth services.

You're All Set!

Your ChatGPT Custom GPT is now connected to MyApi

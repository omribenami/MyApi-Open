2. Service Description

MyApi is a self-hosted AI agent management platform that provides:

  • AI Personas: Create, configure, and manage AI agent personas with custom instructions and knowledge bases
  • Conversations: AI-powered chat sessions using configured personas and connected knowledge
  • Knowledge Base: Upload and manage documents for AI context and retrieval
  • Skills Marketplace: Create, publish, fork, and install custom skill packages
  • OAuth Integrations: Connect third-party services (Google, GitHub, Slack, Discord, and others) for extended capabilities
  • API Access: Programmatic access via REST API with scoped tokens for automation and integration
  • Workspaces: Multi-tenant team environments with role-based access control
  • Device Management: Device fingerprinting and approval workflows for secure access
  • Data Export/Import: Portable data export in ZIP format with integrity verification
  • Agent and Agentic Access: Access via third-party AI platforms and agents including ChatGPT GPTs, MCP (Model Context Protocol) clients, OpenAPI-defined agents, Hermes-compatible platforms, and other automated orchestration systems, through the Service's API
  • AI Routing Infrastructure: Requests to cloud AI providers are intermediated through an AI routing proxy (OpenClaw) that adds processing, logging, and routing logic between your request and the final AI provider
